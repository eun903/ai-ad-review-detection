## AI 기반 광고성 리뷰 탐지 웹 서비스 ✨

### 📖 프로젝트 개요
소비자는 제품 구매 전 리뷰를 주요 판단 기준으로 삼지만,
**광고성 리뷰의 확산**으로 인해 리뷰 신뢰도가 낮아지는 문제가 발생하고있다.
이를 해결하기 위해 AI를 활용하여 리뷰의 **진위 여부를 자동 판별**하는 시스템을 개발하였다.

### 💡 서비스 소개
- 사용자가 리뷰를 입력하거나 이미지를 붙여넣으면(OCR) AI가 **광고성 여부를 분석, 시각화, 요약**까지 자동으로 처리한다.
- **유사도 기반 분석 + 키워드 가중치**를 적용하여 광고 리뷰와 비광고 리뷰를 정밀하게 구분한다.
